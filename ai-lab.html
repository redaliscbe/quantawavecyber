About the Lab

The AI & Cyber Analytics Lab conducts applied research and hands-on development in cyber threat detection, anomaly analytics, adversarial robustness for AI systems, and automated incident response. We bridge theory and practice by working with real telemetry, curated datasets and defensive tooling.

  • Threat detection using ML/AI and telemetry fusion
  • Adversarial machine learning & model robustness
  • Behavioral analytics for insider threat detection
  • SOC automation, alert triage and alert prioritization
Lab Snapshot

Focus: AI for Cybersecurity, Threat Analytics

Resources: GPU Nodes, Packet Capture Cluster, SIEM Access

Open to: Researchers, Students and Industry collaborators

Research Areas

Threat Detection & Hunting

ML-driven detection using network, endpoint and log telemetry; threat hunting playbooks.

Adversarial ML & Robustness

Research into adversarial attacks, defenses, certified robustness and safe model deployment.

Behavioral Analytics

User and entity behavior analytics for insider threat detection and anomaly scoring.

Cloud Security Analytics

Cloud telemetry analysis, misconfiguration detection and identity anomaly detection.

SOC Automation & Orchestration

Automating triage, enrichment, and playbook execution using ML and rule-based systems.

Explainable & Interpretable Models

Model explanations, feature attribution and human-in-the-loop verification for analysts.

Equipment & Tools

GPU Cluster (for model training)

Packet capture & replay cluster

SIEM / Log aggregation & query

Cloud sandboxes & containerized labs

Selected Projects

ML-based Network Intrusion Detection

Hybrid ML models combining flow features and deep packet inspection signatures.

Adversarial Robustness for Malware Classifiers

Benchmarking evasion attacks and defenses on static & dynamic analysis models.

Automated Alert Prioritization

Ranking noisy SIEM alerts using risk scores and analyst feedback loops.
