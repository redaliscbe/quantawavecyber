Program Structure — Combined Certification Package

This is a single combined 6-month academy package. Each vendor certification listed above is delivered as one module. The full program culminates in a capstone project and an elective choice.

Core Modules (Vendor-wise)
  1. Module 1 — Cisco: CCNA Security / ASA / FTD
  2. Module 2 — Palo Alto: PCCET / PCNSA / PCNSE
  3. Module 3 — Fortinet: NSE 3 / NSE 4
  4. Module 4 — Check Point: CCSA / CCSE
  5. Module 5 — Juniper: SRX / JNCIS-SEC
Finals & Electives
  • Capstone Project: Real-world integrated project that spans multiple vendor platforms — presented and evaluated at program end.
  • Elective (choose one):
    • Certified Ethical Hacker (CEH) elective paper
    • OR Internship in our CyberClinic (live incident handling & SOC exposure)
    • OR On-site job training / placement support for Data Centre & SOC roles

Assessment: Module-end practical tests, vendor-aligned mock exams, capstone evaluation and final viva. Certification: QuantaWave Cyber Academy certificate on successful completion, with guidance for vendor certification exams.
