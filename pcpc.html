Professional Certificate Program in Cybersecurity (PCPC)

Industry-Aligned • Hands-on • Network & SOC Focused Cybersecurity Program

Program Overview

The Professional Certificate Program in Cybersecurity (PCPC) is a six-month, industry-aligned program designed to build strong foundational and operational cybersecurity skills for graduates and early-career professionals.

The program emphasizes network security, firewall administration, SOC fundamentals, and threat mitigation, delivered through enterprise-grade security platforms and real lab environments.

PCPC follows a practice-first learning model combining instructor-led sessions, live device configuration, security policy implementation, and applied cybersecurity projects.

Detailed Syllabus

Module 1: Palo Alto Networks – Cybersecurity Practitioner

  • Fundamentals of cybersecurity, threat actors, attack vectors, and kill chain concepts
  • Understanding modern threat landscapes including malware, ransomware, and APTs
  • Enterprise network, endpoint, and application security fundamentals
  • Introduction to cloud security concepts and shared responsibility model
  • Zero Trust architecture principles and policy-based access control
  • Security operations fundamentals and incident response lifecycle

Module 2: Juniper Networks Security

  • Junos OS architecture and core security concepts
  • Configuration of firewall filters, zones, and security policies
  • Network Address Translation (NAT) and access control mechanisms
  • IPsec VPN fundamentals for secure site-to-site and remote access
  • Intrusion detection, traffic monitoring, and security logging

Module 3: Fortinet Network Security

  • Firewall fundamentals and Fortinet Security Fabric overview
  • Threat actors, attack surfaces, and common exploitation techniques
  • Risk assessment, vulnerability management, and basic compliance concepts
  • Introduction to unified threat management (UTM) and policy enforcement

Module 4: Capstone Project – Applied Network & Cybersecurity

  • Identification of real-world network or cybersecurity problems
  • Secure network design incorporating firewalls, VPNs, and access control
  • Implementation of security policies using enterprise security devices
  • Testing, validation, documentation, and technical presentation

Module 5: Industry Internship / On-the-Job Training

  • Exposure to entry-level Security Operations Center (SOC) environments
  • Hands-on experience in firewall monitoring and log analysis
  • Incident reporting, escalation procedures, and documentation practices
  • Industry mentorship, professional ethics, and workplace readiness

Module 6: Applied Cybersecurity Research & Technical Writing

  • Case-based analysis of cybersecurity incidents and operational challenges
  • Identification of applied research or technical problem statements
  • Structured technical writing aligned with IEEE-style documentation
  • Basic research methodology, result interpretation, and conclusions
  • Ethical practices, plagiarism awareness, and citation standards
